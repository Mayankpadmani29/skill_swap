## Project Overview
SkillSwap aims to build a community-driven learning platform where anyone can teach or learn a skill. Users can:  
- Create and join learning sessions.  
- Connect with experts in different fields.  
- Showcase their skills and portfolios.  
- Earn reputation and feedback from the community.  

---

## Features
- **User Registration & Authentication**: Secure login/signup.  
- **Skill Listings**: Add, browse, and search for skills.  
- **Interactive Dashboard**: Personalized learning and teaching stats.  
- **Responsive Design**: Works seamlessly on both mobile and desktop.  
- **Notifications**: Stay updated on skill sessions and interactions.  
- **Profile Management**: Edit personal info, skills, and achievements.  

---

## Tech Stack
- **Frontend**: Next.js, React, Tailwind CSS  
- **UI Components**: Shadcn/UI, Lottie, Framer Motion  
- **Backend (Future)**: Node.js, Express, MongoDB (planned)  
- **Deployment**: Vercel  
- **Media Handling**: Optimized for images and videos
